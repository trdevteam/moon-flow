## Частые вопросы

### Нужна ли регистрация?
Нет, аккаунт не нужен.

### Нужен ли интернет?
Для основных функций — нет. Интернет может понадобиться для восстановления покупок.

### Где хранятся данные?
Данные хранятся локально на вашем устройстве.

### Как восстановить покупки?
MoonFlow → Premium → Restore Purchases.

### Не загружаются подписки/продукты
Проверьте интернет, попробуйте позже и нажмите Restore Purchases.
Если не помогло — напишите нам модель устройства и версию iOS.

### Stealth Mode
Можно скрыть приложение под иконкой калькулятора и открывать по PIN.
