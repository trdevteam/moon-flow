প্রায়শই জিজ্ঞাসিত প্রশ্ন (FAQ)
আমার কি অ্যাকাউন্ট লাগবে?

না। MoonFlow-এ অ্যাকাউন্ট নিবন্ধন প্রয়োজন নেই।

MoonFlow কি ইন্টারনেট লাগে?

মূল ফিচারগুলোর জন্য না। আপনার সাইকেল ট্র্যাকিং অফলাইনে কাজ করে। কেনাকাটা (purchases) পুনরুদ্ধার করতে ইন্টারনেট কেবল তখনই লাগতে পারে।

আমার ডেটা কোথায় সংরক্ষিত হয়?

আপনার স্বাস্থ্য ডেটা আপনার ডিভাইসে লোকালভাবে সংরক্ষিত থাকে।

আমি কীভাবে কেনাকাটা (purchases) পুনরুদ্ধার করব?

MoonFlow খুলুন → Premium → Restore Purchases।

আমি সাবস্ক্রিপশন অপশন দেখতে পাচ্ছি না / প্রোডাক্ট লোড হচ্ছে না

আপনার ইন্টারনেট সংযোগ পরীক্ষা করুন

কয়েক মিনিট পরে আবার চেষ্টা করুন (স্টোর সিঙ্ক হতে সময় লাগতে পারে)

TestFlight / কেনাকাটার জন্য যে Apple ID ব্যবহার করেছিলেন, একই Apple ID ব্যবহার করছেন কি না নিশ্চিত করুন

Restore Purchases চেষ্টা করুন
তবুও কাজ না হলে, আপনার ডিভাইস মডেল এবং iOS ভার্সন উল্লেখ করে আমাদের ইমেইল করুন।

Stealth Mode কীভাবে কাজ করে?

Stealth Mode অ্যাপটিকে ক্যালকুলেটর-স্টাইল আইকনের আড়ালে লুকিয়ে রাখতে পারে। আপনি আপনার PIN দিয়ে MoonFlow খুলতে পারবেন।
