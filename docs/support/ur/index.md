کیا مجھے اکاؤنٹ کی ضرورت ہے؟

نہیں۔ MoonFlow میں اکاؤنٹ رجسٹریشن کی ضرورت نہیں ہے۔

کیا MoonFlow کو انٹرنیٹ چاہیے؟

بنیادی فیچرز کے لیے نہیں۔ آپ کی سائیکل ٹریکنگ آف لائن کام کرتی ہے۔ خریداری بحال (Restore) کرنے کے لیے انٹرنیٹ صرف اسی صورت میں درکار ہو سکتا ہے۔

میرا ڈیٹا کہاں محفوظ ہوتا ہے؟

آپ کا صحت سے متعلق ڈیٹا آپ کے ڈیوائس پر مقامی طور پر محفوظ ہوتا ہے۔

میں خریداری کیسے بحال کروں؟

MoonFlow کھولیں → Premium → Restore Purchases۔

مجھے سبسکرپشن کے آپشن نظر نہیں آ رہے / پروڈکٹس لوڈ نہیں ہو رہے

اپنا انٹرنیٹ کنکشن چیک کریں

چند منٹ بعد دوبارہ کوشش کریں (اسٹور سنک ہونے میں وقت لگ سکتا ہے)

تصدیق کریں کہ آپ وہی Apple ID استعمال کر رہے ہیں جو TestFlight / خریداری کے لیے استعمال ہوئی تھی

Restore Purchases آزمائیں
اگر پھر بھی مسئلہ برقرار رہے تو اپنے ڈیوائس ماڈل اور iOS ورژن کے ساتھ ہمیں ای میل کریں۔

Stealth Mode کیسے کام کرتا ہے؟

Stealth Mode ایپ کو کیلکولیٹر جیسے آئیکن کے پیچھے چھپا سکتا ہے۔ آپ اپنا PIN استعمال کر کے MoonFlow کھول سکتے ہیں۔
