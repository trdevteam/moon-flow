# Política de Privacidade — MoonFlow

**Data de vigência:** 2025-12-20

A MoonFlow foi criada com privacidade como prioridade. Não exigimos conta e buscamos manter seus dados de saúde sob seu controle.

## 1) Quais dados a MoonFlow armazena
A MoonFlow pode armazenar as informações que você inserir no app, como:
- Datas de menstruação e informações do ciclo
- Dados de fertilidade / ovulação
- Dados do modo gravidez (se ativado)
- Sintomas, humor e anotações que você registrar
- Configurações do app (por exemplo: lembretes, preferências)

Esses dados são usados apenas para oferecer as funcionalidades do app.

## 2) Onde seus dados ficam armazenados
A MoonFlow armazena seus dados **localmente no seu dispositivo**.
Os recursos principais não exigem servidor ou conta da MoonFlow.

Observação: dependendo das configurações do seu dispositivo, backups do sistema (por exemplo, backup do iCloud no iOS) podem incluir dados do app. A MoonFlow não controla o comportamento de backup do sistema.

## 3) Conexão com a internet
A MoonFlow não precisa de internet para os recursos principais.
Se você abrir links externos (por exemplo, páginas de ajuda), a internet será necessária apenas para isso.

## 4) Cadastro de conta
A MoonFlow não exige cadastro.

## 5) Anúncios e rastreamento
A MoonFlow foi projetada para ser **sem anúncios**.
A MoonFlow não vende seus dados pessoais.

## 6) Compras no app (Premium)
A MoonFlow pode oferecer compras Premium (assinaturas ou compra vitalícia).
Os pagamentos são processados pela Apple App Store / Google Play. A MoonFlow não recebe seus dados completos de pagamento (como números de cartão). Podemos receber o status da compra (por exemplo, se o Premium está ativo) para desbloquear recursos no seu dispositivo.

## 7) Serviços de terceiros (análises / relatórios de falhas)
A MoonFlow foi criada para funcionar sem compartilhar seus registros de saúde com terceiros.
A MoonFlow não compartilha seus registros de ciclo, sintomas, humor ou anotações com terceiros.
Se usarmos ferramentas de análise ou relatórios de falhas no futuro, atualizaremos esta política explicando o que é coletado e por quê.

## 8) Como excluir seus dados
Como seus dados ficam no dispositivo, você pode removê-los:
- Usando opções de apagar/redefinir no app (se disponíveis), e/ou
- Desinstalando a MoonFlow
