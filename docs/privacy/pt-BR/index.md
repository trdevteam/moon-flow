# Política de Privacidade — MoonFlow

**Data de entrada em vigor:** 2025-12-20

A MoonFlow foi criada com a privacidade como prioridade. Não exigimos conta e procuramos manter os seus dados de saúde sob o seu controlo.

## 1) Que dados a MoonFlow guarda
A MoonFlow pode guardar as informações que introduz na app, tais como:
- Datas de menstruação e informações do ciclo
- Dados de fertilidade / ovulação
- Dados do modo gravidez (se activado)
- Sintomas, humor e notas que registe
- Definições da app (por exemplo: lembretes, preferências)

Estes dados são usados apenas para disponibilizar as funcionalidades da app.

## 2) Onde os seus dados são guardados
A MoonFlow guarda os seus dados **localmente no seu dispositivo**.
As funcionalidades principais não exigem servidor ou conta MoonFlow.

Nota: dependendo das definições do seu dispositivo, as cópias de segurança do sistema (por exemplo, cópia de segurança do iCloud no iOS) podem incluir dados da app. A MoonFlow não controla o comportamento de backup do sistema.

## 3) Ligação à internet
A MoonFlow não necessita de internet para as funcionalidades principais.
Se abrir ligações externas (por exemplo, páginas de ajuda), a internet será necessária apenas para isso.

## 4) Registo de conta
A MoonFlow não exige registo.

## 5) Anúncios e rastreio
A MoonFlow foi concebida para ser **sem anúncios**.
A MoonFlow não vende os seus dados pessoais.

## 6) Compras na app (Premium)
A MoonFlow pode oferecer compras Premium (subscrições ou compra vitalícia).
Os pagamentos são processados pela Apple App Store / Google Play. A MoonFlow não recebe os seus dados completos de pagamento (como números de cartão). Podemos receber o estado da compra (por exemplo, se o Premium está activo) para desbloquear funcionalidades no seu dispositivo.

## 7) Serviços de terceiros (análises / relatórios de falhas)
A MoonFlow foi concebida para funcionar sem partilhar os seus registos de saúde com terceiros.
A MoonFlow não partilha registos de ciclo, sintomas, humor ou notas com terceiros.
Se utilizarmos ferramentas de análise ou relatórios de falhas no futuro, actualizaremos esta política e indicaremos o que é recolhido e porquê.

## 8) Como eliminar os seus dados
Como os seus dados ficam no dispositivo, pode removê-los:
- Usando opções de apagar/repor na app (se existirem), e/ou
- Desinstalando a MoonFlow
