# Politique de confidentialité — MoonFlow

**Date d’entrée en vigueur :** 2025-12-20

MoonFlow est conçue avec la confidentialité comme priorité. Aucun compte n’est requis et nous visons à vous laisser le contrôle de vos données de santé.

## 1) Données stockées par MoonFlow
MoonFlow peut stocker les informations que vous saisissez dans l’application, par exemple :
- Dates de règles et informations de cycle
- Données de fertilité / d’ovulation
- Données du mode grossesse (si activé)
- Symptômes, humeur et notes que vous enregistrez
- Paramètres de l’application (ex. : rappels, préférences)

Ces données sont utilisées uniquement pour fournir les fonctionnalités de l’application.

## 2) Où vos données sont stockées
MoonFlow stocke vos données **localement sur votre appareil**.
Les fonctionnalités principales ne nécessitent pas de serveur ni de compte MoonFlow.

Remarque : selon les réglages de votre appareil, les sauvegardes système (par exemple la sauvegarde iCloud sur iOS) peuvent inclure des données de l’application. MoonFlow ne contrôle pas le comportement des sauvegardes système.

## 3) Connexion internet
MoonFlow ne nécessite pas de connexion internet pour les fonctionnalités principales.
Si vous ouvrez des liens externes (par exemple des pages d’aide), l’accès internet est requis uniquement pour ces liens.

## 4) Création de compte
MoonFlow ne nécessite pas de création de compte.

## 5) Publicité et suivi
MoonFlow est conçue pour être **sans publicité**.
MoonFlow ne vend pas vos données personnelles.

## 6) Achats intégrés (Premium)
MoonFlow peut proposer des achats Premium optionnels (abonnements ou achat à vie).
Les paiements sont traités par l’Apple App Store / Google Play. MoonFlow ne reçoit pas vos informations de paiement complètes (comme les numéros de carte). Nous pouvons recevoir le statut d’achat (par exemple Premium actif) afin de débloquer des fonctionnalités Premium sur votre appareil.

## 7) Services tiers (analytique / rapports de crash)
MoonFlow est conçue pour fonctionner sans partager vos données de santé avec des tiers.
MoonFlow ne partage pas vos données de cycle, symptômes, humeur ou notes avec des tiers.
Si nous utilisons des outils d’analytique ou de rapports de crash à l’avenir, cette politique sera mise à jour pour expliquer ce qui est collecté et pourquoi.

## 8) Suppression de vos données
Comme vos données sont stockées localement, vous pouvez les supprimer en :
- utilisant les options de suppression/réinitialisation dans l’app (si disponibles), et/ou
- désinstallant MoonFlow de votre appareil
