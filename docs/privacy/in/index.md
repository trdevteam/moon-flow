MoonFlow dirancang dengan privasi sebagai prioritas utama. Kami tidak mewajibkan akun, dan kami berusaha agar data kesehatan Anda tetap berada dalam kendali Anda.

## 1) Data apa yang disimpan MoonFlow
MoonFlow dapat menyimpan informasi yang Anda masukkan di aplikasi, seperti:
- Tanggal menstruasi dan informasi siklus
- Data kesuburan / ovulasi
- Data mode kehamilan (jika diaktifkan)
- Gejala, suasana hati, dan catatan yang Anda tulis
- Pengaturan aplikasi (misalnya: pengingat, preferensi)

Data ini hanya digunakan untuk menyediakan fitur aplikasi.

## 2) Di mana data Anda disimpan
MoonFlow menyimpan data Anda **secara lokal di perangkat Anda**.
Fitur inti tidak memerlukan server atau akun MoonFlow.

Catatan: tergantung pengaturan perangkat, cadangan sistem (misalnya cadangan perangkat iCloud di iOS) dapat menyertakan data aplikasi. MoonFlow tidak mengontrol perilaku pencadangan sistem.

## 3) Koneksi internet
MoonFlow tidak memerlukan internet untuk fitur inti.
Jika Anda membuka tautan eksternal (misalnya halaman bantuan), internet hanya diperlukan untuk itu.

## 4) Registrasi akun
MoonFlow tidak memerlukan registrasi akun.

## 5) Iklan dan pelacakan
MoonFlow dirancang **tanpa iklan**.
MoonFlow tidak menjual data pribadi Anda.

## 6) Pembelian dalam aplikasi (Premium)
MoonFlow dapat menawarkan pembelian Premium opsional (langganan atau pembelian seumur hidup).
Pembayaran diproses oleh Apple App Store / Google Play. MoonFlow tidak menerima detail pembayaran lengkap (seperti nomor kartu). MoonFlow dapat menerima status pembelian (misalnya Premium aktif) untuk membuka fitur Premium di perangkat Anda.

## 7) Layanan pihak ketiga (analitik / laporan crash)
MoonFlow dirancang untuk bekerja tanpa membagikan catatan kesehatan Anda kepada pihak ketiga.
MoonFlow tidak membagikan data siklus, gejala, suasana hati, atau catatan Anda kepada pihak ketiga.
Jika kami menggunakan alat analitik atau laporan crash di masa depan, kebijakan ini akan diperbarui untuk menjelaskan apa yang dikumpulkan dan alasannya.

## 8) Cara menghapus data
Karena data disimpan secara lokal, Anda dapat menghapusnya dengan:
- Menggunakan opsi hapus/reset di aplikasi (jika tersedia), dan/atau
- Menghapus instalasi MoonFlow dari perangkat Anda
