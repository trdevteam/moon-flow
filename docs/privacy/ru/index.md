# Политика конфиденциальности — MoonFlow

**Дата вступления в силу:** 2025-12-20

MoonFlow создано с приоритетом конфиденциальности. Аккаунт не требуется, и мы стремимся, чтобы ваши данные о здоровье оставались под вашим контролем.

## 1) Какие данные хранит MoonFlow
MoonFlow может хранить данные, которые вы вводите в приложении, например:
- Даты менструации и информация о цикле
- Данные о фертильности / овуляции
- Данные режима беременности (если включён)
- Симптомы, настроение и заметки, которые вы записываете
- Настройки приложения (например: напоминания, предпочтения)

Эти данные используются только для работы функций приложения.

## 2) Где хранятся ваши данные
MoonFlow хранит данные **локально на вашем устройстве**.
Для основных функций не нужен сервер MoonFlow и не требуется аккаунт.

Примечание: в зависимости от настроек устройства системные резервные копии (например, резервная копия устройства iCloud на iOS) могут включать данные приложения. MoonFlow не контролирует поведение резервного копирования системы.

## 3) Интернет-соединение
MoonFlow не требует интернета для основных функций.
Если вы открываете внешние ссылки (например, страницы помощи), интернет нужен только для этого.

## 4) Регистрация аккаунта
MoonFlow не требует регистрации.

## 5) Реклама и отслеживание
MoonFlow задумано как приложение **без рекламы**.
MoonFlow не продаёт ваши персональные данные.

## 6) Покупки в приложении (Premium)
MoonFlow может предлагать необязательные покупки Premium (подписки или пожизненная покупка).
Платежи обрабатываются Apple App Store / Google Play. MoonFlow не получает полные платежные данные (например, номера карт). Мы можем получать статус покупки (например, активен ли Premium), чтобы разблокировать Premium-функции на вашем устройстве.

## 7) Сторонние сервисы (аналитика / отчёты о сбоях)
MoonFlow создано так, чтобы не делиться вашими записями о здоровье с третьими лицами.
MoonFlow не передаёт третьим лицам данные цикла, симптомы, настроение или заметки.
Если в будущем мы будем использовать аналитику или отчёты о сбоях, мы обновим эту политику, указав, что собирается и зачем.

## 8) Как удалить данные
Поскольку данные хранятся локально, вы можете удалить их:
- используя функции удаления/сброса в приложении (если доступны), и/или
- удалив MoonFlow с устройства
